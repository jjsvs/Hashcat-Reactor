HASHCAT REACTOR

Pebble Time 2 companion settings

Backend

Where your Hashcat Reactor desktop bridge is listening. Use http://localhost:3001 when running the Pebble emulator on the same PC. For real watch use, expose the bridge over your LAN or a tunnel like zrok (see the desktop app's Remote Access panel).
localhost:3001 LAN:3001

Refresh

How often the phone fetches new stats from the bridge. Lower = fresher, higher = less battery drain on the phone.

Auth (optional)

Only needed if you protected the bridge with HTTP Basic auth (e.g. via zrok's --basic-auth flag).
